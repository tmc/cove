# Examples

Scenario-based tutorials showing how to use cove for real-world workflows.

- [macOS CI Runner](ci-runner.md) -- Run automated tests in disposable macOS VMs with snapshot-based rollback.
- [Reproducible Dev Environment](dev-environment.md) -- Define a complete development environment with vzscripts and shared folders.
- [Security Research Sandbox](security-sandbox.md) -- Analyze untrusted code in isolated, disposable VMs with network controls.
- [Node.js MCP Client](nodejs-mcp-client.md) -- Drive cove from TypeScript over the Model Context Protocol: list VMs, screenshot, and exec.
